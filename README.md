**A distilled blogging engine for [goholic.in](https://goholic.in).**

Malt is an exercise in restraint. It is a production-grade content API written in Go 1.22+ without frameworks.

* **No Gin, No Echo:** Just `net/http` and `serveMux`.
* **No ORM:** Just raw SQL and `modernc.org/sqlite`.
* **No Build Step:** Just `go run` and Vanilla JS.
